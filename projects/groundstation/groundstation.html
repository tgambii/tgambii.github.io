Infrastructure

PDM, BOM & Data Ownership.

Before this project, no formal CAD data management system existed. Autodesk Vault was implemented from scratch as part of the THC redesign — creating the engineering data infrastructure the team now runs on. Managing it on an ongoing basis is part of the role.

🗂️
Autodesk Vault PDM — implemented & administered
Built the full system from zero: folder structure, file lifecycle states (Work In Progress → Released), formal check-in/check-out workflows, user access permissions, and revision history. Ongoing administration is a sole responsibility.
📊
SAP BOM — created & maintained
Multi-level bill of materials in SAP ERP covering the full assembly — part numbers, quantities, cost codes, and revision flags. Built during design and maintained through each ECO to keep production data current and accurate.

How It Came Together

Design to Deployment.

PDM infrastructure was established before design work began — so every file, revision, and release had a formal home from day one. Design and data management ran in parallel throughout.

1
Requirements
Interface specs & design inputs
Captured military radio interface requirements, environmental constraints, and installation workflow targets. Primary KPI: reduce field installation time and eliminate on-site engineering dependency.
2
Infrastructure
Autodesk Vault implementation
Stood up the PDM system before design began — folder structure, lifecycle states, check-in/out discipline, and team permissions configured so every file is formally tracked from the first commit.
3
Concept Design
Layout generation & down-select
Generated multiple configuration layouts in CAD. Down-selected on producibility, assembly access, and serviceability. Top-level interface dimensions locked before detail design began.
4
Detail Design
Full parametric 3D model
Built the complete assembly — all detail parts, sheet metal components, and hardware. Revision discipline maintained in Vault throughout. SAP BOM entries created in parallel as parts were finalized.
5
Drawing Release
Production drawing package
Authored detail, assembly, and installation drawings per ASME Y14.5 with GD&T on all critical interfaces. Managed ECO revision cycles through formal Vault release workflows.
6
Production & Deployment
Vendors, BOM & field documentation
Coordinated fabrication vendors and delivery schedules for on-time completion. Authored the illustrated field assembly guide — enabling the first fully remote installation of this product.
